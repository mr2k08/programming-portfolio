# Projects

A selection of personal programming projects in **C**, **C++**, and **Python**.

## C / C++

- **C - Neural Network from scratch (MNIST)** — a feed-forward neural network written in plain C, trained on the MNIST handwritten-digit dataset.
- **C++ - LobsterLearn (neural net library)** — a small neural-network library in C++ (matrix, layers, activations, network).
- **C++ - Space Invaders** — an Atari-style Space Invaders game.
- **C++ - Trading Backtester** — a backtesting engine for trading strategies.
- **C++ - Pong / Snake / Spinning Donut** — classic graphics/game demos.
- **C - Snake** — Snake implemented in C.

## Python

- **Python - Claw (AI coding agent)** — a command-line AI coding agent (a Claude Code–style clone).
- **Python - Larry (AI agent)** — a tool-using AI agent.
- **Python - YouTube Player (PyQt6)** — a desktop YouTube player with a multi-module PyQt6 GUI.
- **Python - Chess (incl. blind chess)** — a chess engine and game, including a blindfold variant.
- **Python - Pokemon game** — a Pokémon-style game with a roster/database.
- **Python - Tic-Tac-Toe (with bot + UI)** — Tic-Tac-Toe with an AI opponent and a graphical interface.
- **Python - Paint app** — a simple drawing application.
- **Python - Buyer-Seller ML simulation** — a machine-learning buyer/seller market simulation.
